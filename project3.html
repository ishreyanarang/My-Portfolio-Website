SMS Spam Filtering System

A machine learning-based web application developed to detect and filter out spam SMS messages. This system uses Natural Language Processing (NLP) techniques to analyze text messages and classify them as either "spam" or "ham" (legitimate). It provides a simple and intuitive interface for users to test and understand how spam detection works in real-time.

Key Features:

  • Real-time spam vs. ham message classification
  • Text input for custom SMS analysis
  • Model trained on real-world SMS datasets
  • Clean, interactive, and responsive UI
  • Displays prediction results with accuracy metrics

Tech Stack:PowerShell,CSS,Python,Shell,JavaScript,HTML,Batchfile
