AI Text Summarizer App

An intelligent tool that summarizes long pieces of text into concise, meaningful summaries using the power of AI. This project integrates OpenAI's GPT API with Postman to automate and test text summarization workflows. Ideal for reducing reading time while retaining core ideas, it showcases the practical application of AI in content management.

Key Features:

  • Accepts long-form text as input
  • Generates short, coherent summaries in seconds
  • API-based implementation tested via Postman
  • Customizable summary length and tone (informative, formal, etc.)
  • Useful for students, content writers, and researchers

Tech Stack: Postman, OpenAI API, JSON, RESTful APIs
