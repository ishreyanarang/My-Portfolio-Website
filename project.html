Hostel Management System

A full-stack web application designed to streamline hostel administration and student management. This system simplifies tasks such as room allotment, student registration, complaint tracking, and fee management. It includes separate portals for admin and students, ensuring secure access and real-time data updates. Developed with a focus on efficiency and user experience, it reduces manual workload and improves operational accuracy within hostel environments.

Key Features:

  • Student and admin login panels
  • Room allotment and vacancy tracking
  • Online complaint registration and resolution
  • Automated fee management and payment tracking
  • Clean, responsive user interface

Tech Stack: HTML, CSS, JavaScript
